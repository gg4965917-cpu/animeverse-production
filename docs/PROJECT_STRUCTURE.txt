📋 ЯК КОРИСТУВАТИСЯ ЦІЄЮ ЗБІРКОЮ:

1️⃣ ЛОКАЛЬНЕ ТЕСТУВАННЯ:
   npm install                      ← Встановити залежности
   npm start                        ← Запустити API (порт 3000)
   npm run bot                      ← Запустити Bot (потребує TOKEN)
   npm run dev                      ← Запустити Frontend (порт 5173)

2️⃣ ОТРИМАТИ TELEGRAM BOT TOKEN:
   - Напиши @BotFather в Telegram
   - Команда: /newbot
   - Скопіюй TOKEN
   - Вставь у .env: TELEGRAM_BOT_TOKEN=YOUR_TOKEN

3️⃣ ДЕПЛОЙ:
   - Railway для API (api-server.js)
   - Vercel для Frontend (src/)
   - Railway для Bot (telegram-bot.js - запуститься автоматично)
